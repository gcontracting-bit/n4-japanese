N4 Verb Forms

The conjugations that make or break N4 — potential, volitional, conditionals, passive, causative, causative-passive.

First, know your verb group. Every rule below depends on it.

る-verbs (Ichidan)

Stem + る. Drop る to conjugate.
食べる, 見る, 起きる, 寝る

う-verbs (Godan)

End in a -u sound; the vowel before it shifts.
行く, 飲む, 話す, 買う, 待つ

Irregular

Only two — memorise them.
する → し…, 来る(くる) → こ…

▶Potential — "can do"L13

Says you are able to do something. The object often takes が instead of を.

GroupRuleExample
る-verbdrop る → +られる食べる → 食べられる
う-verbfinal -u → -eru行く → 行ける / 飲む → 飲める
する→ できるする → できる
来る→ 来られる (こられる)来る → 来られる
Note: potential verbs all become る-verbs, so they conjugate like 食べる (食べられます, 食べられない, 食べられた).
日本語が話せます。 I can speak Japanese.
きのうは寝られませんでした。 I couldn't sleep yesterday.

▶Volitional — "let's / I shall"L15

The casual form of ～ましょう. With ～と思っています it states an intention.

GroupRuleExample
る-verbdrop る → +よう食べる → 食べよう
う-verbfinal -u → -ou行く → 行こう / 飲む → 飲もう
する→ しようする → しよう
来る→ 来よう (こよう)来る → 来よう
週末、映画を見よう。 Let's watch a movie this weekend.
日本で働こうと思っています。 I'm thinking of working in Japan.

▶Conditional ば — "if"L22

A hypothetical "if A, then B". Focuses on the condition itself.

TypeRuleExample
る-verbdrop る → +れば食べる → 食べれば
う-verbfinal -u → -eba行く → 行けば
い-adjectivedrop い → +ければ安い → 安ければ
negative～ない → ～なければ行かない → 行かなければ
安ければ買います。 If it's cheap, I'll buy it.
薬を飲めば、よくなりますよ。 If you take the medicine, you'll get better.

▶Conditional たら — "when / if"L17

Built straight off the past (た) form — the easiest conditional. "Once A happens, B."

BaseRuleExample
any verbpast た/だ + ら食べた → 食べたら / 飲んだ → 飲んだら
い-adjectivepast かった + ら高かった → 高かったら
noun/な-adjだった + ら暇だった → 暇だったら
うちに帰ったら、電話します。 When I get home, I'll call you.
お金があったら、旅行したいです。 If I had money, I'd want to travel.

▶Passive (受身) — "is done to"L21

The doer takes に. Often used for annoyance ("suffering passive").

GroupRuleExample
る-verbdrop る → +られる食べる → 食べられる
う-verbfinal -u → -areru飲む → 飲まれる / 言う → 言われる
する→ されるする → される
来る→ 来られる (こられる)来る → 来られる
Watch out: for る-verbs the passive and potential look identical (食べられる). Context tells them apart.
先生にほめられました。 I was praised by the teacher.
弟にケーキを食べられた。 My little brother ate my cake (on me).

▶Causative (使役) — "make / let do"L22

Make or let someone do something. The person made/allowed takes に or を.

GroupRuleExample
る-verbdrop る → +させる食べる → 食べさせる
う-verbfinal -u → -aseru行く → 行かせる / 待つ → 待たせる
する→ させるする → させる
来る→ 来させる (こさせる)来る → 来させる
母は妹に部屋をそうじさせた。 Mom made my sister clean the room.
先生は学生を早く帰らせた。 The teacher let the students go home early.

▶Causative-Passive (使役受身) — "made to do"L23

You are made to do something (usually unwillingly). Build causative first, then make it passive: させる → させられる.

GroupRuleExample
る-verbdrop る → +させられる食べる → 食べさせられる
う-verb-u → -aserareru (often contracted -asareru)飲む → 飲まされる / 待つ → 待たされる
する→ させられるする → させられる
来る→ 来させられる (こさせられる)来る → 来させられる
Tip: for う-verbs the contracted form (飲まされる) is very common in speech. ～す verbs like 話す don't contract → 話させられる.
パーティーで歌を歌わされた。 I was made to sing at the party.
子どものとき、毎日ピアノを練習させられた。 As a kid, I was made to practice piano every day.

▶Honorific & Humble verbs (keigo)L19–20

Raise the other person (honorific / 尊敬語) or lower yourself (humble / 謙譲語).

PlainHonorific (them)Humble (me)
いる／行く／来るいらっしゃるおる／参る (まいる)
食べる／飲む召し上がるいただく
言うおっしゃる申す (もうす)
するなさるいたす
見るご覧になる拝見する (はいけんする)
General patterns: honorific お＋stem＋になる (お書きになる); humble お＋stem＋する (お持ちします).

▶Commands — なさい & imperativeL22

なさい is a firm but polite command (parent/teacher → child): verb stem + なさい.

早く寝なさい。 Go to bed early.
よく勉強しなさい。 Study hard.
